日记02

首先学习

总的来说,今天学的很顺,很通透越来越感觉写程序就和搭积木一样,只是现在某些部件不太熟,
还有就是如何更省力,但大体上今天是十分充实的一天Run
其次之前觉得程序报错没关系,只要能跑就行的想法,有很大问题,还是必须去搞懂每个bug的具体
原因,不然之后的连锁反应会让你想死[这个时候就该跑路了doge]

1.pinia过了一遍,感觉跟vuex差不多[因为大概就是vuex.5]

pinia安装: npm i pinia -s
1.新建vuexStore/index.js
2.main.js引入
3.书写


import { defineStore } from “pinia”;

export const storeA = defineStore("storeA", {...});
export const storeB = defineStore("storeB", {...});
export const storeC = defineStore("storeB", {...});

要准备洗澡了,还回顾了axios和mock,为什么今天会去回顾主要刚把登录页面写完,写其他的一点都不通畅,

因为之前都是照着视频或者文档写的,自己一写就乱套了,不太好,效率太低,回炉重造.呸
